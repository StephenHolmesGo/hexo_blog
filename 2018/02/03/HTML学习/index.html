html简介

HTML 指的是超文本标记语言，首先，HTML不是一种编程语言，而是一种标记语言，特别适合没有任何基础的"小白"学习，而且很简单，学习HTML不仅可以让你知道怎么编写一个网站，也是打开你通往黑客世界的一扇门，让你了解代码，让你慢慢熟悉黑客世界这个环境
